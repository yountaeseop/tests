1. 깃 저장소 만들기
-> git init 
2. 현재 git 상태보기
-> git status
3. 저장소에 저장하고 싶은 파일을 인덱스에 먼저 올리기
-> git add 파일명
4. 인덱스에 올라간 파일을 갓 저장소에 반영
-> git commit-m"메시지"
5. github.com가입

6. 깃 유저 정보 등록(github 가입 정보로 등록)
-> git config --local user.name "github nickname"
-> git config --local user.name "github email"

7. git 저장소 원격저장소에 업로드
-> git push 원격지 주소 master
